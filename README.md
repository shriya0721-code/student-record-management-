# Student Management System (Python CLI Project)

## 📌 Objective
This project is a menu-driven Python program that manages multiple students and performs basic academic analysis.  
The system runs continuously until the user chooses to exit.

## 🚀 Features

- Add a new student
- Store marks for 3 subjects
- Calculate average marks
- View all students
- Search student by name
- Display topper (highest average)
- Continuous menu-driven execution

## 🛠 Concepts Used

- Variables
- Lists
- Dictionaries
- Loops (for, while)
- Conditional Statements
- Functions
- Built-in functions (sum, max, len)
- Lambda function
- String handling

## 📂 Data Structure Used

Students are stored in a list of dictionaries.
